# OCaml
My OCaml codes for the programming contest 

### Opam installations 
Opam is OCaml package manager tool.
For Mac 
$ brew install opam 
$ opam install omake 
$ eval $(opam env)

### OMake installation and build 
For Mac
$ omake --install 
Make OCaml .exe file  
$ omake
Execute it 
$ ./main.exe
