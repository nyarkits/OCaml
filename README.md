# OCaml
My OCaml codes for the programming contest 

### Opam installations 
<p> Opam is OCaml package manager tool. </p>
<p> For Mac </p>
<p> $ brew install opam </p>
<p> $ opam install omake </p>
<p> $ eval $(opam env) </p>

### OMake installation and build 
<p> For Mac </p>
<p> $ omake --install </p>
<p> Make OCaml .exe file </p>
<p> $ omake </p>
<p> Execute it </p>
<p> $ ./main.exe </p>
